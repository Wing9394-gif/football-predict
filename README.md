# 足球博彩预测系统 v2.1

> 泊松 + 蒙特卡洛 + 多源数据 + 资金流 + 错价比对 + Dutching 对冲波胆 + 赛后自动复盘

## 架构概览

```
系统自动抓取捷报数据
        ↓
   ┌────┴────┐
   │ 亚盘模型 │         │ 大小球模型 │
   │ (独立)   │  ←→     │ (独立)     │
   └────┬────┘         └────┬────┘
        │                   │
        └──────┬────────────┘
               ↓
         交叉验证
               ↓
   ┌──────┴──────┐
   │  波胆筛选   │
   │  (Dutching) │
   └──────┬──────┘
          ↓
  亚盘 + 大小球 + 波胆 三路交叉验证
          ↓
      最终推荐
```

## 核心能力

### 一：亚盘预测
- 零堆积双变量泊松（ZIBP）构建进球概率模型
- λ 多源6因子：α近期进球 β历史交锋 γxG δ进攻提速 ε转换率 η关键球员
- 五维特征加权：赔率隐含35% / 基本面20% / 历史20% / 阵容15% / 战意10%
- 盘口对冲 + 欧亚转换校验 + 资金流监控
- **赔率门槛 ≥ 1.75（皇冠基准）**

### 二：大小球预测（独立模型）
- 多因子整合：xG/xGA + 进攻提速(Pace) + 转换率(Conversion) + 伤停 + 联赛因子 + 半场分布 + 角球密度 + 主客场 + 裁判风格
- 蒙特卡洛万次模拟 → AI理论进球线(Theoretical Line)
- 错价比对（≥0.25球筛选核心）
- **赔率门槛 ≥ 1.75（皇冠基准）**

### 三：波胆筛选 + Dutching
- xG异常值检测 + 核心复出加权偏移
- 筛选3个EV>0的波胆
- Dutching反推对冲本金分配

### 四：三路交叉验证
- 亚盘 ↔ 大小球（矛盾标记）
- 亚盘 ↔ 波胆（方向一致性）
- 大小球 ↔ 波胆（总进球范围）
- 三路一致 → 高置信度推荐

### 五：EV全覆盖
- 每个投注方向全部计算 EV
- **只推荐 EV > 0 的方向**

## 数据源

### 主数据源
- 捷报比分网 `nowscore.com`

### 辅助数据源
- **Tier 1:** Opta/StatsBomb xG、FBref/Understat xG、Betfair Exchange
- **Tier 2:** OddsPortal/FlashScore盘口变动、Transfermarkt/WhoScored伤停、ClubElo
- **Tier 3:** 天气、排名、官方新闻

## 赛后自动复盘（四步流程）

```
Step 1: 偏差分析（比分/亚盘/大小球/xG → ✅❌⏸️标记）
Step 2: 异常信号复盘（赛前信号 vs 实际比赛 → 虚警率计算）
Step 3: 参数在线更新（π/λ/αβγδεη/δ/阈值/权重 → 旧值→新值）
Step 4: 累计准确率追踪（写入记忆文件，🎯 亚盘≥70% 大小球≥70%）
```

## 系统流程

1. **工作流一：赛前数据采集** — 自动抓取捷报比分网数据（盘口/欧赔/基本面/阵容/伤停/裁判）
2. **工作流二：预测分析** — 三路独立运算 + 交叉验证 + EV全覆盖
3. **工作流三：赛后复盘** — 四步自动复盘 + 参数学习 + 记忆持久化
