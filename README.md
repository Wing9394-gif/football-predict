# Football Predict v2.5

足球博彩预测系统 — 泊松 + 蒙特卡洛 + 多源数据 + 错价比对 + Dutching 对冲波胆

## 核心架构

```
捷报数据 → 亚盘模型 ↔ 大小球模型 → 交叉验证 → 波胆筛选 → 最终推荐
```

## 三大工作流

1. **赛前数据采集** — 捷报比分网实时盘口/基本面/伤停/裁判/教练
2. **预测分析** — 零堆积双变量泊松 + 蒙特卡洛 10,000~100,000 次模拟
3. **赛后复盘** — 偏差分析 → 异常信号复盘 → 参数在线更新 → 准确率追踪

## v2.5 更新

- 🏷️ **粤语译名规则** — 所有球队/球员/教练名统一使用港澳粤语译音（普通话译音，英文原名）
- 💰 **赔率门槛修正** — 亚盘/大小盘使用净赔率（不含本），≥0.80 方予推荐；欧赔含本 ≥1.80
- 📐 **净赔率明确区分** — 皇冠亚盘/大小盘为净赔率（@1.02 = 投100净赚102）
