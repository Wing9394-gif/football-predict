# ⚽ 足球博彩预测系统 v2.3

基于泊松分布 + 蒙特卡洛模拟 + 多源数据融合的足球博彩预测系统。

## 核心能力

- **赛前数据采集**：自动抓取捷报比分网盘口、分析、裁判、教练数据
- **亚盘预测**：泊松模型 + 多因子加权 + 欧亚转换校验 + 资金流错价检测
- **大小球预测**：多因子独立运算模型
- **波胆 Dutching**：对冲策略，多比分覆盖
- **赛后复盘**：偏差分析 → 异常信号追踪 → 参数在线更新 → 准确率追踪
- **中立场地系数**：世界杯/杯赛自动修正客场龙/客场虫差异

## 架构

```
数据采集 → 亚盘模型(独立) ←→ 大小球模型(独立) → 交叉验证 → 波胆Dutching → 三路统一输出
```

## 参数 (v2.3)

| 参数 | 值 | 说明 |
|------|:--:|------|
| γ (xG权重) | 0.30 | 淘汰赛xG预测力 |
| η (关键球员) | 0.12 | 核心伤停权重 |
| 淘汰赛大小球修正 | -0.10 | 杯赛进球修正 |
| 中立场地系数 | 1.05/0.93 | 客场龙/客场虫 |
