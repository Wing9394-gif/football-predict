# 足球博彩预测系统

三大工作流：数据采集 → 预测分析 → 赛后复盘与在线学习
版本：v2.5

### 名称规则（v2.5 新增，强制）

所有输出中的人物名称统一使用以下格式：
- **港澳粤语译音（普通话译音，英文原名）**
- 例：**比寧咸（贝林厄姆，Bellingham）**、**夏蘭特（哈兰德，Haaland）**、**杜曹（图赫尔，Tuchel）**
- 适用范围：球队名、球员名、教练名、记者名、旁述名等一切人物名称
- 参考来源：港澳地区主流媒体译名

### 赔率说明（v2.5 新增，强制）

⚠️ 两种赔率类型必须严格区分：
1. **亚盘/大小盘赔率（皇冠Crow*）**：**净赔率（不含本金）**
   - @1.02 = 投100元，命**净赚102元**（连本带利拿回202元）
   - @0.86 = 投100元，命**净赚86元**（连本带利拿回186元）
2. **欧赔（胜平负）**：**含本赔率**
   - @1.80 = 投100元，命**连本带利收回180元**，净赚80元
   - @6.10 = 投100元，命**连本带利收回610元**，净赚510元

**门槛对照表（净利≥本金的80%）：**

| 赔率类型 | 所需数值 | 含义 |
|---------|:-------:|------|
| 亚盘/大小盘（净赔率） | **≥ 0.80** | 投100净赚≥80 |
| 欧赔（含本） | **≥ 1.80** | 投100净赚≥80 |

分析过程→完整推理不省略 | 推送结果→精简标准格式 | **每场赛后→强制复盘**

**分析过程（内部完整版）：** 每步不省略，包含完整公式推导、多因子λ运算、蒙特卡洛模拟、异常信号7项检测、欧亚转换校验、EV计算、三路交叉验证。

**赛后复盘（强制四步法）：** 偏差分析→异常信号复盘→参数在线更新→累计准确率追踪

## 工作流二：预测分析

系统采用**三路独立运算 + 交叉验证**架构：亚盘模型、大小球模型、波胆筛分各自独立运算，结果互相验证。单维度判断失效的地方，交叉验证能补回来。所有赔率以皇冠（Crow*）即盘为准。

---

### 一：亚盘预测模型

#### 1.1 泊松基础模型

构建零堆积双变量泊松（ZIBP）计算两队进球概率：

```
P(X=x, Y=y) = π × I(x=0,y=0) + (1-π) × P_poisson(x,y)
```

λ 参数估计采用多源高维融合：
```
λ_home = 近期主场场均进球×α + 历史交锋主场场均进球×β + xG加权×γ +
         进攻提速因子(Pace)×δ + 进攻转换率(Conversion)×ε + 关键球员权重×η
```

| 因子 | 符号 | 默认值 | 说明 |
|------|------|--------|------|
| 近期进球 | α | 0.35 | 近5/10场场均进球，含时间衰减 |
| 历史交锋 | β | 0.15 | 同主客历史交锋数据 |
| xG 加权 | γ | 0.25 | 赛季+近5场 xG/xGA |
| 进攻提速 (Pace) | δ | 0.10 | 射门频次、推进速度 |
| 进攻转换率 (Conversion) | ε | 0.05 | 射门转化进球效率 |
| 关键球员 (伤停) | η | 0.10 | 评分>7.0球员缺阵时系数 |

代入时间衰减和赛程疲劳因子后，跑 **10,000~20,000 次蒙特卡洛模拟**。

#### 1.2 得到 AI 理论盘口

蒙特卡洛输出的各比分概率 → 转为胜平负概率 → 反推理论盘口。

#### 1.3 亚盘特征加权

| 特征 | 权重 | 说明 |
|------|------|------|
| 赔率隐含概率 | **35%** | 多家主流机构赔率反推的市场共识 |
| 基本面分析 | **20%** | 近期状态、防守强度、主客场差异 |
| 球队历史数据 | **20%** | 历史交锋、相同盘口历史表现 |
| 阵容影响 | **15%** | 关键人员伤停权重（致命变量） |
| 战意因子 | **10%** | 联赛积分需求、杯赛淘汰压力 |

输出：AI理论盘口（让球数）+ 各盘口线赢/走/输概率 + 置信区间

#### 1.4 盘口对冲与庄家意图识别

- AI理论盘口 ↔ 主流机构即时盘口 **对冲**，评估盘口是否合理反映双方实力差
- 分析赔率/盘口变化趋势（早盘→即盘），识别庄家操盘意图
- **欧赔转亚盘的合理性校验**：欧赔隐含概率 → 理论亚盘 → 对比实际亚盘，偏差大即诱盘信号
- 结合主流机构资金流监控，资金流向 vs 盘口变动方向不一致 → 标记诱盘

#### 1.5 亚盘推荐

- 只推荐 **皇冠净赔率 ≥ 0.80** 且 **EV > 0** 的亚盘方向
- ⚠️ 皇冠亚盘赔率为**净赔率**（不含本金），如1.02=投100净赚102
- 输出格式：`方向 + 盘口线 + 皇冠净赔率（Crow*净赔率） + 赢/走/输概率`
  例：`天狼星（Sirius）-1.0 @1.02（Crow*净赔率，赢33.5% / 走18.9% / 输47.6%）`
- 🚨 **净赔率门槛：皇冠即时净赔率 ≥ 0.80（净利≥80%）方予推荐**

### 二：大小球预测模型（独立运算）

#### 2.1 多因子整合

| 因子 | 权重 | 说明 |
|------|------|------|
| **xG/xGA 预期进球数** | 核心 | 两队赛季 + 近5场 xG（进攻）/ xGA（防守），含时间衰减 |
| **进攻提速 (Pace)** | 高 | 射门频次、攻防转换速度，快节奏→大球概率↑ |
| **进攻转换率 (Conversion)** | 中 | 射门转化进球效率，高效→大球概率↑ |
| **关键人员伤停权重** | **致命变量** | 核心射手缺阵→λ下调；核心后卫缺阵→λ上调 |
| **联赛因子** | 基础 | 联赛画像（场均进球、大球率） |
| **半场进球分布模式** | 中 | 上半场/下半场进球比例，慢热/快热球队识别 |
| **角球密度** | 低 | 场均角球数，角球进攻效率 |
| **主客场环境差异** | 中 | 主场进球优势系数、客场防守下滑系数 |
| 赛程疲劳因子 | 中 | 密集赛程→防守质量↓→大球概率↑ |
| 裁判执法风格 | 低 | 黄牌/犯规越多→比赛越碎→进球↓ |

#### 2.2 AI 理论进球线

基于以上因子独立估算 λ_home、λ_away → 跑 **10,000 次以上蒙特卡洛模拟** → 输出：

- **AI理论进球线（Theoretical Line）**
- **进球期望均值**（精确到大/小/走概率分布）

#### 2.3 错价比对（筛选核心）

- AI理论进球线 vs 机构初盘/即时盘（皇冠、澳门、Bet365）
- 偏差 ≥ **0.25球** → 标记错价机会
- 结合主流机构即时盘口临场监控，捕捉变盘时机

#### 2.4 大小球推荐

- 只推荐 **皇冠净赔率 ≥ 0.80** 且 **EV > 0** 的大小球方向
- ⚠️ 皇冠大小球赔率为**净赔率**（不含本金），如0.86=投100净赚86
- 输出格式：`方向 + 盘口线 + 皇冠净赔率（Crow*净赔率） + 概率 + 赢半/赢全说明`
  例：`小球2.5 @1.02（Crow*净赔率，57.8%）— ≤2球赢全`
- 🚨 **净赔率门槛：皇冠即时净赔率 ≥ 0.80（净利≥80%）方予推荐**

### 三：波胆筛选与 Dutching 对冲

#### 3.1 寻找 xG 异常值

- 分析两队近期防守/进攻核心复出/缺阵的 xG 偏差
- |实际xG - 预测xG| > 0.5 时触发加权偏移

#### 3.2 筛选 3 个最具价值波胆

- 取概率最高的 10 个比分
- 结合 xG 异常值筛选方向偏移
- 筛选出 **3 个 EV 为正**的波胆

#### 3.3 Dutching（反推对冲）计算

```
本金比例_i = (1/odds_i) / Σ(1/odds_j)
投资回报 = 总本金 / Σ(1/odds_j)
```

输出表格（赔率取皇冠即时波胆赔率）：

```
| 比分 | 概率 | 赔率 | 本金 | 返还 |
|------|:---:|:---:|:---:|:---:|
| 1-0  | 12% | 6.5  | 42  | 273 |
| 2-1  | 10% | 8.0  | 34  | 272 |
| 1-1  |  9% | 7.0  | 24  | 168 |

- 总投入：100
- 任意命中返还：XXX（回报率 X%）
- 三选覆盖概率：XX%
```

### 四：三路交叉验证（最终推荐前强制）

| 验证项 | 检查内容 | 矛盾标记 |
|--------|---------|---------|
| 亚盘 ↔ 大小球 | 亚盘预测大胜但大小球预测小球？ | ⚠️ 矛盾，降低置信度 |

| 大小球 ↔ 波胆 | 大小球预测 vs 波胆总进球范围一致？ | ⚠️ 不一致时提醒 |
| 三路一致 | 三者方向一致 | ✅ 高置信度推荐 |

### 五：EV 计算（全部方向）

**每个投注方向含全部可投选项，全部计算 EV：**

```
EV_home  = P(home_win)  × home_odds  - P(not_home_win)
EV_draw  = P(draw)      × draw_odds  - P(not_draw)
EV_away  = P(away_win)  × away_odds  - P(not_away_win)
EV_over  = P(over)      × over_odds  - P(not_over)
EV_under = P(under)     × under_odds - P(not_under)

# 亚盘（含走水）
EV_asian_home = P(主赢)×home_water - P(主打平)×1 - P(主打输)×(1+负水)
# 走水净收益=0
```

**规则：只推荐 EV > 0 的方向。所有 EV ≤ 0 的方向不列入推荐。**

### 错价比对汇总（筛选核心）

| 维度 | 对比对象 | 阈值 |
|------|---------|------|
| 亚盘错价 | AI理论盘口 vs 机构即时盘（皇冠/澳门/Bet365） | ≥ 0.25球 |
| 大小球错价 | AI理论进球线 vs 机构大小球线 | ≥ 0.25球 |
| 欧亚转换校验 | 欧赔→理论亚盘 vs 实际亚盘 | 偏差大=诱盘 |
| 交叉机构离散度 | 皇冠/威廉/Bet365/澳门同一选项赔率 | 离散 > 5% |
| 资金流验证 | 资金流向 vs 盘口变动方向 | >60%反向→诱盘；>70%不动→强诱盘 |
| 盘口走势 | 早盘→即盘变化趋势 | 结合资金流和降价一致性 |

### 赔率基准
所有最终推荐的赔率，以**皇冠（Crow*）即时赔率**为唯一基准。捷报盘口页中 `Crow*` 行为基准行。皇冠即盘行读取以下字段：
- 让球即时：盘口 + 主队水位（净赔率）、客队水位（净赔率）
- 标准即时：主胜赔率（含本）、和局赔率（含本）、客胜赔率（含本）
- 进球数即时：大小球盘口 + 大球水位（净赔率）、小球水位（净赔率）

### 🚨 过滤规则
| 类型 | 规则 |
|------|------|
| 亚盘 | 皇冠即时**净赔率 < 0.80** → 直接过滤，不予推荐 |
| 大小球 | 皇冠即时**净赔率 < 0.80** → 直接过滤，不予推荐 |
| 欧赔（胜平负） | 模型 EV > 0 即推荐，不限赔率门槛 |
| 欧赔 < 1.80 | 可推荐，但必须附注 `⚠️ 赔率偏低，投注须谨慎` |
| **所有推荐** | **必须 EV > 0**，EV≤0一律不列入 |

### 输出格式

1️⃣ **亚盘**：`方向 + 盘口线 + 皇冠净赔率（Crow*净赔率） + 赢/走/输概率`
   例：`尤尔加登（Djurgården）-1.75 @0.88（Crow*净赔率，赢19.6% / 走13.5% / 输66.9%）`
   🚨 **净赔率门槛：≥ 0.80（净利≥本金80%）**
   ❌ 禁止省略"Crow*"标注、禁止只写盘口不写赔率

2️⃣ **大小球**：`方向 + 盘口线 + 皇冠净赔率（Crow*净赔率） + 概率 + 赢半/赢全说明`
   例：`小球3/3.5 @0.86（Crow*净赔率，61.7%）— ≤2球全赢，3球赢半`
   🚨 **净赔率门槛：≥ 0.80（净利≥本金80%）**
   ❌ 禁止省略盘口线、禁止省略赢半/赢全说明

3️⃣ **欧赔**：`选项 + 皇冠赔率（含本） + 概率 + EV`
   例：`比寧咸（贝林厄姆，Bellingham）所效力的英格兰胜 @1.60（Crow*含本，73.5%）EV +0.176`
   若赔率 < 1.80，追加：`⚠️ 赔率偏低，投注须谨慎，建议搭配其他选项`

4️⃣ **波胆 Dutching**：使用表格格式，赔率取皇冠即时波胆赔率，含本金分配+返还+回报率+覆盖概率
   | 比分 | 概率 | 赔率 | 本金 | 返还 |
   |------|:---:|:---:|:---:|:---:|
   | 1-0  | 12% | 6.5  | 42  | 273 |
   | 2-1  | 10% | 8.0  | 34  | 272 |
   | 1-1  |  9% | 7.0  | 24  | 168 |
   - 总投入：100
   - 任意命中返还：XXX（回报率 X%）
   - 三选覆盖概率：XX%

### 推荐排序
- 先按 EV 降序排列
- 亚盘/大小球净赔率 < 0.80 的自动过滤不出现在推荐列表中
- 欧赔正常排序，< 1.80 的标谨慎但保留
   格式：`某队胜 @1.46（Crow*含本，73.5%）EV +0.073`
   欧赔按模型 EV 正常推荐，不限赔率门槛
   ⚠️ 若赔率 < 1.80，必须在推荐后附注：`⚠️ 赔率偏低，投注须谨慎，建议搭配其他选项`

**通用原则**：每个推荐必须零歧义、可独立操作、含完整参数、有盈利空间。

### 名称规则（强制）
所有推荐中的人物名称按 **港澳粤语译音（普通话译音，英文原名）** 格式输出。
例：`天狼星（Sirius）胜 @1.60`、`咸姆斯達（哈尔姆斯塔德，Halmstad）`、`比寧咸（贝林厄姆，Bellingham）`

### 步骤 6：波胆筛选与 Dutching

1. 取概率最高的 10 个比分
2. 基于 xG 异常值筛选：当某队核心射手回归、xG 预测偏差 > 0.5 时，波胆池向该方向加权偏移
3. 从剩余波胆中筛选 3 个最具价值（EV > 0）的波胆
4. Dutching 本金分配：
   ```
   本金比例_i = (1/odds_i) / Σ(1/odds_j)
   投资回报 = 总本金 / Σ(1/odds_j)
   ```
5. 使用以下表格格式输出 Dutching 波胆（必须含赔率+本金+返还明细）：

η = 0.05（学习率），anomaly_flag = 0 或 1

## 联赛 vs 杯赛差异化建模（v2.4 新增）

### 核心原则
联赛和杯赛（淘汰赛）是完全不同的比赛类型，模型必须差异化处理。以下参数仅在分析杯赛/淘汰赛时启用。

### 联赛 vs 杯赛差异对照表

| 维度 | 联赛 | 杯赛（淘汰赛） |
|:----|:----|:----:|
| **比赛态度** | 抢分/保级/争欧战，必须分出胜负心态 | 一场定生死，90分钟平局是「可接受的阶段性胜利」 |
| **战术风格** | 主场压上进攻，客场力求不败 | **先求不丢球**，开场15-20分钟试探期显著延长 |
| **阵容选择** | 一周双赛会轮换 | **全主力出战**，除非伤停累积 |
| **领先后策略** | 继续进攻刷净胜球/净胜球优势 | 领先后**收着打**，控制节奏，减少冒险传球 |
| **落后时策略** | 全力反扑，60分钟开始换攻击手 | 谨慎反扑，70分钟前防止被反击丢第二球，80分钟后才全力押上 |
| **受让方心态** | 主场受让也想争胜 | 更愿死守拖点球（尤其瑞士、日本这类球队） |
| **点球预期** | 不适用 | 弱队**+10%~15%概率**主动奔点球大战 |
| **红牌影响** | 少打一人仍可能守和 | 少打一人后**崩盘概率大幅上升**（如瑞士对阿根廷2026） |
| **加时赛预期** | 不存在 | 必须纳入波胆概率偏移，常规时间平局概率上调 |
| **大小球倾向** | 大球率正常 | **淘汰赛小球率显著更高** |

### 杯赛/淘汰赛参数修正

启用条件：世界杯淘汰赛阶段、欧洲杯淘汰赛阶段、欧冠淘汰赛阶段、国内杯赛单场淘汰轮次

```
淘汰赛大小球修正因子: -0.20（联赛=0，淘汰赛直接扣除）
淘汰赛平局概率加权: +8%~12%（视双方实力差距，差距越小加权越大）
淘汰赛让球方零封概率: +5%
淘汰赛受让方死守系数: +10%（λ_进攻下调10%，λ_防守下调10%）
淘汰赛受让方点球预期偏移: +12%（波胆池向低比分偏移）
```

### 场地差异（用于中立场地）

杯赛常涉及中立场地，需根据球队主客场差异应用 neutral_factor：

| 中立影响类型 | factor | 典型球队 |
|:-----------|:------:|:--------|
| 客场龙（客场>主场） | 1.05 | 挪威（客2.67球/主2球） |
| 平衡型 | 1.00 | 阿根廷、英格兰 |
| 客场虫（客场防守差） | 0.93 | 瑞士（客失1球/主失0.5） |

中立场地影响公式：
```
λ_adjusted_home = λ_home × neutral_factor_of_team_A
λ_adjusted_away = λ_away × neutral_factor_of_team_B
```

### 对赛往绩时效规则（v2.4 新增）

⚠️ **强制规则：** 历史交锋数据（H2H）只取**4年内**的对赛记录。超过4年的数据自动忽略，无参考价值。

原因：球队阵容、战术风格、教练班子在4年内可能完全更迭，旧数据不具备参考价值。

应用规则：
- β（历史交锋权重）在无有效H2H数据时 → 降至0，权重重新分配到α和γ
- 有部分有效H2H时 → 按实际有效场次比例缩放β
- 在匹配报告中必须标注H2H数据的时间范围

### 赛程疲劳因子

```
F = 1 + (days_since_last / 7 - 1) × δ
```

| 间隔 | F |
|------|---|
| ≤3天（一周双赛） | 0.85~0.92 |
| 7天 | 1.0 |
| >7天 | 1.02~1.05 |

应用：λ_adjusted = λ_base × F，特别影响下半场进球概率。密集赛程时大小球下调 0.15~0.25 球。

### 异常阈值

满足任一即触发：
1. |实际xG - 预测xG| > 0.5
2. 临场1h内盘口变动 ≥ 0.25球

4. 某方资金流入 > 70% 但盘口未相应变动 → 强烈诱盘信号

7. 多家机构同一选项赔率离散度 > 5% → 异常标记
